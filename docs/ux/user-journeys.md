# User Journeys

## First Slice

1. Start the application.
2. Choose a PDF.
3. See the first page render.
4. Run extraction.
5. Review highlighted words on the PDF.
6. Select a word to inspect geometry and confidence.
7. Save the project.
8. Reopen the project and continue review.

## UX Principles

- Keep the recommended next action visible.
- Keep extracted values synchronized with their PDF locations.
- Show confidence using text plus visual state, not color alone.
- Avoid technical extraction settings before a first useful result.
